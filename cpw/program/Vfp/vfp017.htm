指定一个数，在另外12个数中，求出最接近（近似等于）这个数的组合



解：
    我想以一个非常有趣的数列为例，来叙述这个问题的解决方法。数列 1，1，2，3，5，8，13，21，24，55，89，144，233，377，610，987…任选其中的相邻的两项An与An+1, 可得到分数An/An+1.。n的值越大，分数An/An+1的值越接近一个重要的数：（√5 -1）/2。这个数是个宝贵的数--黄金分割数，程序中取的变量名为GOLD。我们任取12个这样的分数，求最接近于黄金数的一个组合。
    建立一个数据库，其名为SJK.dbf它有四个字段，其中cn 是An/An+1,的值，由程序中的语句repl all cn with a1/a2 自动求出。 
另一字段 ABS_N_0是An/An+1与黄金数之差的绝对值。程序中由语句repl all abs_n_0 with abs(cn- gold) 求出。为什么要取绝对值？这个问题留给读者琢磨。以下是数据库的内容。

即是最接近黄金分割数的一个序列。

算法（程序）如下：
SET DECIMAL TO 8
use sjk

gold= (sqrt(5)-1)/2
repl all cn with a1/a2
repl all abs_n_0 with abs(cn- gold)

sort on abs_n_0 to sjkpx
use sjkpx
list
