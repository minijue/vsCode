1、自动加上最后修改日期
    我们在 HTML文件完成了以后，常会加上一行网页最后修改的日期，现在你可不用担心每次都要去改或是忘了改了。你可以很简单地写一个如下的描述语法程式，就可以自动地为你每次产生最后修改的日期了：
    <html>
    <body>
    本页最后修改日期为
    <br>
    Last changes:
    <script language=“LiveScript">
    <!－－ hide script from old browsers
    document.write(document.lastModified)
    // end hiding contents －－>
    </script>
    </body>
    </html>

    2、按钮的设置
    当您在“中国软件信息港”进行网上购物时，可以选定欲购物品，再点一下“放入购物车”，屏幕     上立刻会显示“商品已放入购物车”，这是如何实现的呢？
    <html>
    <head>
    <script language=“LiveScript">
    function pushbutton() {
    alert("商品已放入购物车"); }
    </script>
    </head>
    <body>
    <form>
    <input type=“button" name=“Button1" value=“放入购物车" onclick=“pushbutton()">
    </form>
    </body>
    </html>

    3、改变超级链接时状态行上的显示
    当鼠标移至普通链接上的时候，会在状态栏显示该链接的目的地，但有时并不希望这样，比如当鼠     标移至此链接时，会在状态栏中显示“此链接将指向《中国电脑教育报》的主页上”，其实只要在传统     <a>的标签中加入onMouseOver的method，就可达到需要的效果了。

    <a href=“http://www.cce.com.cn" onMouseOver=“window.status=‘此链接将指向《中国电脑教     育报》的主页上'; return true">
