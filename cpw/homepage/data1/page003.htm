十足动感，编织梦幻---DreamWeaver2动态主页设计



　　Frontpage问世之后，由于罩着Microsoft的光环并伴随着IE的“血本大甩卖”，它迅速成为网页设计的主流软件。尽管人们对其有太多的抱怨也不得不凑合着使用，同时又在瞪大眼睛寻找着“新欢”。这时，DreamWeaver2出现了！
　　DreamWeaver2这个“梦幻编织者”是顺应动态主页设计（DHTML）的潮流诞生的，目前最新为2.1版本，由大名鼎鼎的Macromedia开发。其令人目眩的动态页面设计、方便直观的层功能、良好的浏览器适应性使你切实体会到制作一个Killer级的主页是多么简单！
　　打开DreamWeaver2（图1），可见这个Killer级的设计软件界面也是不俗的，但如果你没有较好的主页制作和Javasc-ript基础的话，要掌握这些按钮和窗口还得用点功才行。

OBJECTS(对象)工具条
　　OBJECTS工具条主要包含插入页面的各类对象，如表格和图片等。（图2）
　　插入图片无须赘述；在插入表格时输入表格的列数、行数和间距等参数即可。为了使主页在高分辨率下面看起来正常，建议锁定表格尺寸，即在Width后面的单位中选择Pixel而不选择Percent。
　　设置表格底色等属性可以利用界面下方的属性窗口。当我们选中一个对象时，窗口便显示出该对象的所有属性（包括该对象的超级链接属性，缺省为“#”即没有超级链接）供编辑修改。例如我们点击表格边沿选中表格，然后单击最下面一行的“BG”后面的按钮，弹出一个调色板窗口选择表格背景色（上面一个“bg”为设置背景图）。现在试着将鼠标移到调色板窗口外的任何位置，你定会大吃一惊，甚至移到菜单栏和标题栏时都可以采样色彩！
　　拆分、合并单元格的方法是在单元格内点右键，然后选TABLE下面的相应命令即可。
　　第三个按钮为插入直线，取消其属性窗口中的“SHADING”可以去掉阴影。
　　接下来的“层”对象是DreamWeaver2的特色之一，利用它可以在主页中轻松插入和编辑层，而无须象以前那样完全靠手工输入代码。分层显示在现在的主页中是相当流行的，我们可以在页面的同一个地方叠上许多层然后再分别控制各层的显现和运动，这就克服了传统主页在同一个地方只能有一个对象而且页面单调的弊病。分层是大部分动态设计的基础，在后面我们将看到,利用层功能我们可以做出许多很专业的特效。
　　现在，在页面中我们已经插入了一幅图片，想要图片上面再放一张图或者其他对象，这就必须用到分层。单击对象工具条上的“插入层”按钮，然后在图片上面的某个地方拖动鼠标画出一个框便定义了一个层，现在我们就可以在这个层中插入任何对象和链接(图3）。如果还嫌不够，可以在该层的上面再插入一层，直至满足需要。
　　很自然我们立即会遇到这样一个问题:“一个地方有几层，怎样规定在浏览时显示哪一层呢？”DreamWeaver2提供了方便的层显示/隐藏控制，按F11键打开层对象窗口就可以设置层的显示或隐藏,缺省的设置是将后插入的层放在上面(图4）。
　　工具条上的其它对象在此不作一一介绍，最后的“Rollvoer Image”主要用来制作两幅图实现鼠标感应的动态按钮。点击Original Image按钮，在弹出对话框的Original Image中选择初始图片，在Original Image中选择当鼠标放在该图片上时显示的第二幅图片，然后设好图片对象名和链接即可（没有链接则保留“#”）。存盘预览一下，一个动感十足的按钮就这么轻松搞定！省去了繁杂的代码编写工作。

LAUNCHER工具条
　　还是用来启动或关闭相应的功能窗口，这些窗口提供了一些DreamWeaver2的核心功能（图5）。现择其要者阐述。
　　站点管理:DreamWeaver2的站点管理功能类似Frontpage98的Explore，可以十分方便地管理站点和自动修改、检查链接。点击站点管理（SITE）按钮打开站点管理窗口，先选择“Define Sites...”定义一个站点（图6）。
　　在设置窗口中选择“Local info”添入站点名;接下来的“Local Root”是网站在本机上的目录;“Link Management Options”是用来检查页面中可能包含链接向远程服务器端对象的超级链接，请在此输入你的网站URL地址。接着再选中“cache”后面的检查框。
接下来设置服务器信息“Web Server Info”，选择“none”表示只在本地机修改;“Local/Network”表示Web服务器在本地局域网，这种情况下设好服务器的局域网地址即可；“FTP”表示通过FTP上传主页，此种情况要设定FTP服务器信息和帐号口令。
　　另外，如果你从事小组式的网站开发，为了防止一个页面同时被多人打开修改，建议你选中“check in/out”中的“enable file check in and check out”并输入你的名字，以后当你编辑某个页面时，网站管理会显示此页面正在被你打开。
　　库:制作网站时经常会遇到这样的情况:有些元素会在许多页面中存在，而且其链接和相关代码都完全相同，如网站的目录图片。利用“库”功能可以很方便地解决这个问题。DreamWeaver2为每个网站建立一个目录来专门存储这些重复利用的对象。
　　首先单击“Library”按钮打开库窗口，向库中增添一个元素的方法是选中该对象拖到库窗口的第一个小窗口之内。这样我们以后在编辑此网站的其他页面要插入此对象时，只需从库中选中并拖入即可。最方便的是当我们修改了库中的对象时，插入了该对象的所有页面会自动地相应更新（包括该对象涉及的链接和相应的代码）。如果我们要对从“库”中插入的对象进行一些特殊修改，可以先选中它，然后在属性窗口中单击“Make Editable”后进行修改，不过这样做后该对象将不再与库映射，今后库中的对象修改后它不会自动更新。
　　行为:如果你对JavaScript或VBScript比较熟悉的话，相信对WEB页涉及的对象、事件、属性等也不会陌生。利用DreamWeaver2我们可以很简单地实现许多以前只能靠手工输入代码才能实现的功能。从某种意义上说，有了DreamWeaver2，一般用户几乎没有必要去学习脚本语言。下面我们以用Behavior方式实现前面的Rollover Image动态按钮为例来说明其用法。
　　在页面中插入图片，然后先选中该图片，单击Behavior打开窗口，点击“+”增加一个行为（DreamWeaver提供了相当丰富的行为，几乎可以满足各种需要），这里我们选择“Swap Image”即交换图片。（图7）
　　在弹出窗口（图8）中单击“Browse”选择想要变换为的第二张图片。
　　下面的检查框“Preload Image”表示在调入页面时就载入第二张图片，“Restore Image OnMouseOut”表示当鼠标移开时自动恢复为初始图片。选中二者点击“OK”。此时我们看到Behavior窗口中增加了两个行为：OnMouseOver（即鼠标停留在该图片上时）对应的行为是Swap Image即变换图片。
　　然而刚才我们并没有设置什么时候（即发生什么事件时）进行该行为，这里的OnMouseOver是缺省的事件，可以任意修改。例如我们想要鼠标点击时变换图片而不是当鼠标移到该图片上时变换，我们可以选中OnMousever然后点击后面的小三角选择OnClicK即可。另一个行为OnMouseOut时“Swap Image Restore”是由于我们前面选中了“Restore Image OnMouseOut”自动增加的。
　　Behavior的功能是相当丰富的，除了可以判别浏览器类别、控制层的隐藏/显示、改变其他对象属性、判断有没有一些插件等等外，Dream-Weaver2还支持用户自定义的行为。我们在为对象增加行为时选择“Call JavaScript”就可以调用用户自己定义的JavaScript函数（定义方法为点击对象工具条上面的小三角，然后选择“Invisible”对象组，再单击“Insert Script”即可）。灵活应用Behavior可以做出交互性和动感很强的主页。如果您对Behavior窗口提供的事件和行为不很了解，建议去查查有关JavaScipt的书。
　　TimeLine(时间轴):如果前面介绍的一些动态页面功能还不能让您为之心动的话，TimeLine定会让你体会一下“三月不知肉味”的感觉，好东西总在最后，下面就将你的鼠标放上去Press一下吧！
　　如果您曾经使用过Flash、3DMAX或者一些动画软件，相信你对该窗口的作用一定早有预感，不错，这是用来做动画的，它会让你的页面真正跳动起来！下面我们就来看看吧。
　　TimeLine动画主要针对层，它可以动态改变层的位置、大小（只有IE4支持）、图片对象的SRC属性等。首先在页面中建立一个层，在其中放入一些对象，如插入一张图片，接着点击层边沿选中该层（注意不要选成了图片），选择Modify菜单下的“Add Object To Timeline”，我们会发现在Timeline1的时间轴上多了一条直线，线上的两个圆点表示的关键帧，点击一下最后一个圆点选择第二个关键帧，将层拖到另一个位置，你发现了什么？多了一条直线！这正是播放时该动画的运动轨迹。
　　现在选中“Autoplay”和“loop”前面的检查框，存盘预览一下，你将切实体会到DHTML那个“D”的妙处！
　　如果想要让层沿曲线运动，可以选中两个关键帧中的某一帧，选择Modify/Timeline/ AddKeyframe插入一个关键帧，然后按住Ctrl键改变层的位置，可以看到层的运动轨迹变成了一条自动拟合的曲线。
　　另一个方法是先选中层，然后选择Modify/Record Path Of Layer，然后拖动层任意移动即可。
　　我们可以建立任意多的Timeline，而且可以向每个Timeline随意增加层，修改各个关键帧等等。限于篇幅请朋友们自己去实践。
　　接下来介绍一下对动画的播放控制，前面我们都是将播放设置为自动重复播放的，其实在Behavior的行为列表中提供了对Timeline动画的播放、停止、跳转等控制，十分方便。例如我们想在点击层中的图片时该层才开始播放，可先选中该图片，打开Behavior窗口，在增加行为时选择Timeline下的Play Timeline。
　　行为和时间轴两个窗口是动态主页设计的核心，也是笔者最欣赏的功能。DreamWeaver2提供的对动态HTML的支持功能也是非常强大的，限于篇幅就不一一详细介绍。
